# Бушуев Никита БАС-1 Лабораторная №1

## Задача 1

### Текст задачи

Получить список из последних цифр чисел, содержащихся в исходном списке

### Алгоритм решения

1.Запрашиваем у пользователя количество элементов n
2.Объявляем рекурсивную функцию readElements с параметрами acc (накопитель) и count (счетчик)
3.Если count = 0, возвращаем перевернутый аккумулятор List.rev acc
4.Иначе запрашиваем элемент x, добавляем его в начало аккумулятора (x::acc) и рекурсивно вызываем функцию с count - 1
5.Запускаем readElements [] n и получаем исходный список
6.Объявляем функцию lastDigit = x % 10 для нахождения последней цифры
7.Применяем List.map lastDigit к исходному списку для получения списка последних цифр
8.Выводим полученный список на экран

### Тестирование

Скриншоты результата работы программы

## Задача 2

### Текст задачи

Найти сумму тех элементов списка, которые оканчиваются на заданную цифру

### Алгоритм решения

1.Запрашиваем у пользователя количество элементов n
2.Объявляем рекурсивную функцию readElements с параметрами acc (накопитель) и count (счетчик)
3.Если count = 0, возвращаем перевернутый аккумулятор List.rev acc
4.Иначе запрашиваем элемент x, добавляем его в начало аккумулятора (x::acc) и рекурсивно вызываем функцию с count - 1
5.Запускаем readElements [] n и получаем исходный список
6.Запрашиваем у пользователя цифру d (от 0 до 9), на которую должны оканчиваться элементы
7.Применяем функцию List.fold к исходному списку с начальным значением аккумулятора 0: для каждого элемента x если последняя цифра числа x равна d (x % 10 = d), то добавляем x к аккумулятору, иначе оставляем аккумулятор без изменений
8.Выводим результат на экран

### Тестирование

Скриншоты результата работы программы
